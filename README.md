# DMProfiles

A Python package for plotting and fitting DM matter profiles.  Currently supports isothermal sphere, PIEMD, NFW, and tNFW profiles.
See Appendix of [Kneib & Natarajan (2011)](https://link.springer.com/article/10.1007/s00159-011-0047-3)
and [Baltz, Marshall, and Oguri (2009)]([https://link.springer.com/article/10.1007/s00159-011-0047-3](https://arxiv.org/abs/0705.0682))

This README will be updated with more information.
